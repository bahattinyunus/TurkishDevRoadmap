# ☁️ DevOps Yol Haritası

Yazılım (Dev) ve Operasyon (Ops) ekiplerini birleştiren kültür ve araçlar bütünü.

## 1. Güçlü Linux Temeli
- Sunucu yönetimi, bash scripting, izinler, dosya sistemleri.

## 2. Konteynerizasyon (Containerization)
- **Docker**: Uygulamayı bağımlılıklarıyla paketleme. "Benim makinemde çalışıyordu" sorununa son.
- **Docker Compose**: Çoklu konteyner yönetimi.

## 3. CI/CD (Sürekli Entegrasyon / Dağıtım)
- **Jenkins**: Eski toprak ama çok yaygın.
- **GitLab CI / GitHub Actions**: Modern ve entegre çözümler.
- **Mantık**: Kod pushlandığında otomatik test et, build al ve sunucuya gönder.

## 4. Orkestrasyon
- **Kubernetes (K8s)**: Konteynerlerin yönetim platformu. DevOps'un zirvesi.

## 5. Cloud (Bulut) Sağlayıcılar
- **AWS (Amazon)**: Pazar lideri.
- **Azure (Microsoft)**: Kurumsalda yaygın.
- **Google Cloud (GCP)**: Yapay zeka ve big data işlerinde güçlü.

## 6. Infrastructure as Code (IaC)
- **Terraform**: Altyapıyı kod ile kurma ve yönetme.
- **Ansible**: Konfigürasyon yönetimi.

## Türkiye Piyasası Notları
> DevOps, şu an Türkiye'de ve dünyada maaş skalası en yüksek alanlardan biridir. Ancak genellikle "Junior DevOps" ilanı azdır. Genellikle Backend veya Sistem kökenliler DevOps'a evrilir.
